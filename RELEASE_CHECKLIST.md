## 应急预案

### 1. 回滚方案
```bash
# Android 回滚
# 1. 下架当前版本
# 2. 重新上架旧版本

# 网页端回滚
# 1. 切换 CDN 到旧版本
# 2. 恢复数据库备份
```

### 2. 故障处理
| 故障类型 | 处理方案 | 负责人 |
|---------|---------|--------|
| 服务器宕机 | 切换备用服务器 | 运维 |
| 数据库故障 | 恢复备份 | DBA |
| CDN 故障 | 切换备用 CDN | 运维 |
| API 异常 | 降级服务 | 开发 |
